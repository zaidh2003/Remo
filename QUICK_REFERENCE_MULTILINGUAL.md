## 🔄 How to Change Language

### Step-by-Step

1. **Find the Globe Icon** 🌐
   - Located in the sidebar header
   - Next to the RestaurantOS logo

2. **Click the Globe Icon**
   - Dropdown menu appears
   - Shows all 3 languages with flags

3. **Select Your Language**
   - Click on your preferred language
   - UI updates immediately
   - Selection is saved automatically

4. **Done!**
   - All text now appears in your language
   - No page refresh needed
   - Works on all pages

## 🔧 Troubleshooting

### Language Not Changing?

**Problem**: UI doesn't update when I select a language

**Solutions**:
1. Refresh the page (F5 or Ctrl+R)
2. Clear browser cache
3. Try a different browser
4. Contact IT support

### Some Text Still in English?

**Problem**: Parts of the UI are still in English

**Solutions**:
1. Check if you're on a new page that just loaded
2. Wait a moment for the page to fully load
3. Refresh the page
4. Report missing translations to IT

### Language Resets to English?

**Problem**: Language changes back to English after closing browser

**Solutions**:
1. Check browser settings allow localStorage
2. Don't use private/incognito mode
3. Check browser extensions aren't blocking storage
4. Contact IT support
