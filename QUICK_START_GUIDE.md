# Quick Start Guide - Fix Missing Branch in Profile

## The Problem
Your profile shows **"Branch: —"** because existing users don't have branches assigned yet.

## The Solution (3 Easy Steps)

### Step 1: Login as Admin
- Use your admin account to access the dashboard

### Step 2: Go to Branch Management
- Click **"Branches"** in the sidebar navigation
- You'll see the Branch Management page

### Step 3: Click "Assign Branches to Users"
- Look for the **blue button** at the top right that says **"Assign Branches to Users"**
- Click it
- Confirm the action in the dialog
- Wait a few seconds for completion
- The page will automatically refresh

## What Happens Next

✅ All users without branches will be assigned to "Urmo Projects" (or your first branch)  
✅ Your profile will now show the branch name instead of "—"  
✅ All branch-related features will work correctly  
✅ You can reassign users to different branches later via User Management

## Troubleshooting

**Q: Button is disabled/grayed out?**  
A: Wait a moment - it's processing. Should complete in 5-10 seconds.

**Q: Still showing "—" after clicking?**  
A: Refresh the page manually (F5 or Ctrl+R)

**Q: No branches exist?**  
A: The system will automatically create "Urmo Projects" branch for you

**Q: Want to change branch assignments?**  
A: Go to User Management → Edit user → Change branch field
